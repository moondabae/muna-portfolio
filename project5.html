Merdeka Memory Match Game

Interactive Card Matching Game

Description: Merdeka Memory Match Game is an interactive game designed to celebrate Malaysia's Independence Day by engaging players in a fun memory match challenge featuring iconic Malaysian symbols.

Key Features

  • Multiple difficulty levels: Easy, Medium, Hard.
  • Timed gameplay to challenge memory speed.
  • Interactive and engaging UI with Malaysian themes.
  • Real-time score tracking and performance feedback.
  • Responsive design for accessibility on various devices.

Technologies Used

  • GDevelop 5 for game development
