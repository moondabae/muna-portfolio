Pop Culture Quiz: Anime & K-Pop Edition

A Fun and Interactive Web Game

Description: Pop Culture Quiz: Anime & K-Pop Edition is an engaging and interactive web game designed to test your knowledge of anime and K-Pop culture. Players can answer multiple-choice questions and earn points based on their accuracy and speed. The game features vibrant visuals and dynamic animations to enhance the user experience.

Key Features

  • Multiple-choice questions on popular anime and K-Pop topics.
  • Timer-based scoring system for added challenge.
  • Responsive design for seamless gameplay on different devices.
  • Leaderboard to track top scores.

Technologies Used

  • HTML, CSS, JavaScript
